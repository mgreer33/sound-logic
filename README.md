# sound-logic
SWEN 661 Project repo for the team Sound Logic

Care connect is an application where individuals with disabilities can request aid at an increased frequency and decreased difficulty. This is to supplement their disabilities and allow individuals to get a tailored experience based on their particular ailments. Our group will focus specifically on hearing loss in this aspect.


Run the app by using the main function in main.dart 

Run the tests by using flutter test --coverage or by running the main function in main_test.dart. That will execute the entire test suite.
